# Website CMS

Website CMS Base on Egg Framework

## QuickStart

```shell
$ npm install
$ npm run dev
$ open http://localhost:7001/news
```

```shell
$ npm run cms:dev
```

```shell
$ npm run dev
$ open http://localhost:7001/news
```
